# CLAUDE.md

This repository uses the same baseline rules as [AGENTS.md](AGENTS.md).

## Short Instruction Set

1. Respect project scope: outbound delivery operations only.
2. Avoid overengineering; prefer simple, auditable implementations.
3. Preserve safety defaults (rate limits, suppression, unsubscribe compliance).
4. Do not add AI attribution trailers to commit messages.
5. If requirements are unclear, ask one concise question instead of guessing high-risk behavior.
